Comment télécharger le logo :

  1. Choisissez le logo qui vous plaît le plus parmi les trois options
  2. Cliquez sur le bouton "Télécharger" sous le logo
  3. L'image sera téléchargée au format PNG
  4. Vous pouvez ensuite utiliser ce logo pour vos documents, cartes de visite, en-têtes, etc.

Conseil : Le logo 1 (bleu) est le plus polyvalent et fonctionne bien sur la plupart des supports.
